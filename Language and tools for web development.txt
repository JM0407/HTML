HTML and css basics
1. open new folder on vscode
2. create index.html file
3. insert all necessary elements inside the html file
	a.<!DOCTYPE html>
	b.<html></html>
	c.<head></head>
	d.<body></body>
4. to use css, <style></style> under header
5. create class for each tag you wanna separate ex <p class="name">
6. inside style, create either tags or class attributes , ex p{} or .names{}
7. to validate html go to validator.w3.org
8. for css validator go to jigsaw.w3.org/css-validator
